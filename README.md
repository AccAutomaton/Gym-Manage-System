### Based on [gymxmjpa](https://github.com/liujianview/gymxmjpa)

- Web Framwork: Spring Boot 2.0.3
- JDK: 1.8
- Builder：Maven
- ORM：Sping Data JPA
- SQL：MySql 8.0.32
- Security Framwork：Shiro
- Frontend：JSP, LayUI, BootStrap, ECharts
